A subnet is a logical partition of an IP network into multiple, smaller network segments. It is typically used to subdivide large networks into smaller, more efficient subnetworks.

Subnets
	- Dividing the network is called subnet
	- Any IP 192.168.1.0 i.e., id ends with '0' is called Network ID
	- 192.168.1.255 is Broadcast ID

	- Usually our IP Address never end with 0.
	- Our IP Addresses will fall between 192.168.1.1 to 254


 Usually 8bits=1 byte
 32-bit  subnet masks are like:
	255.255.255.255
	  8     8     8    8 =32
