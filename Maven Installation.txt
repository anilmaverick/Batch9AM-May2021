
1. Prerequisites to install maven
	- JDK & JRE should install and configure

edit the system environment variables  system variables
         JAVA_HOME = C:\Program Files\Java\jdk1.8.0_131
         JRE_HOME=C:\Program Files\Java\jre1.8.0_131
        path = C:\Program Files\Java\jdk1.8.0_13\bin 

2. Download Maven 
	https://maven.apache.org/download.cgi

3.After setup the environment variables 
	EnvironmentVariables->user variables
		M2_HOME=C:\apache-maven-3.6.1
		MAVEN_HOME=C:\apache-maven-3.6.1
	SystemVariables
		PATH=C:\apache-maven-3.6.1\bin

	- Open command prompt c:\mvn -version
	(it displays the maven version, which indicates that maven install and configured)
